Toggle or drive snaf-flow mode (iterative step-by-step development without upfront planning).

- `/snaf-flow on` — enable flow mode (same as typing `flow`)
- `/snaf-flow off` — disable flow mode (same as typing `stop flow`)
- `/snaf-flow <cel>` — enable flow mode AND start with the given goal
- `/snaf-flow` (no args) — show current flow mode state

Use the `snaf-flow` skill for all guidance on how the mode behaves (propose one step, wait, execute, report, propose next).
